Hydroshare WMS Assignment

Details

This page is an example of importing data from Hydroshare via WMS. Here is the resource used.

There should be 3 layers: 1 point layer, 1 polyline layer, and 1 polygon layer. However, Hydroshare is currently going through major updates (Feb 2025) and may not display all of the data yet.

Code Explanation

To display a layer via web map service (WMS), the following code can be added to your javascript:

L.tileLayer.wms('https://geoserver.hydroshare.org/geoserver/HS-c647cbb66048497dbca360cfc7ac4863/wms', { layers: 'HS-c647cbb66048497dbca360cfc7ac4863:100ft_faults', format: 'image/png', transparent: false, attribution: 'Hydroshare GeoServer' }).addTo(map);

The link and "layers" addresses must be changed of course, but that's the basic format. Unfortunately, the symbology can't be changed when using data imported by this method.
