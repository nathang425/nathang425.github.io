Engineering Economics

Calculations

Most FE problems on this subject are some variation of "benefits-costs" calculation, adjusted for how value changes over time.

Time Value of Money

You must account for the change in money's value over time. Use the following formulas to find the present or future value of costs given the following variables:

FV = Future Value
PV = Present Value
A = Annuity (payment)
i = interest rate per period (as decimal)
n = number of periods

FV, Single Payment: FV = PV*(1+i)^n

PV, Single Payment: PV = FV/[(1+i)^n]

Future Value of an Annuity: FV = A*([(1+i)^n]-1)/i

Present Value of an Annuity: FV = A*(1-[(1+i)^-n])/i

Capital Recovery Factor: CRF = i*[(1+i)^n]/([(1+i)^n]-1)


Analysis Methods

Using the previous equations, there are several methods of analysis for a particular problem:

Benefit-Cost Analysis: BCR = PV(costs)/PV(benefits)

Present Worth Analysis: NPV = PV(costs)+PV(benefits)

Annual Worth Analysis: AW = NPV*CRF

Rate of Return Analysis: NPV = 0 = (-initCost)+[(annualReturn)*(1-[1+IRR]^-n)/IRR]
